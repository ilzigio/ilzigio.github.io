Autonomous Driving: Road Sign Recognition

This is the final project of the Computer Vision course, the aim of the project was to build a system able to recognize the road sign in a defined set of images. The project has been developed using C++ and the OpenCV library, to recognize the sign are used the classic method of the computer vision and to computer the speed in the speed limit sign an OCR are used. Embedded Systems Programming: Multishot Photography

This is the final project of the Embedded Systems Programming, the aim of the project was to build an application for Android smartphone that implemented an algorithm of multi-shot photography. This project has been developed together with two colleagues, using the Java Android Language for the behavior of the application, while to developed the computational photography algorithm we used C++ languages, the OpenCV library and the Android NDK library to embody the C++ code in the application code. Distributed Systems: Distributed File System

In this project we implement a basic distributed file system using Java and in particular the library Java RMI. The aim of the project was to design and implement a distributed file system with single node fault tolerance. The project is composed by a communication layer used to handle the communication between the host in the distributed file system and a user interface layer, that permits to the user to create, modify, delete a file or a folder.
