Embedded Systems Programming: Multishot Photography

This is the final project of the Embedded Systems Programming, the aim of the project was to build an application for Android smartphone that implemented an algorithm of multi-shot photography. This project has been developed together with two colleagues, using the Java Android Language for the behavior of the application, while to developed the computational photography algorithm we used C++ languages, the OpenCV library and the Android NDK library to embody the C++ code in the application code.
